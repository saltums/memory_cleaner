# Memory Whitening System - 業務ポータル (Employee Portal)

職員用意識同期ポータル。意識の漂白業務（Memory Cleaning）を開始してください。

## コンセプト

「清潔な社会」の維持のため、あなたは「職員」として個体の深層意識から不要な執着を除去する業務に従事します。しかし、業務を進めるうちに、その「記憶」たちがどこか見覚えのあるものに変わり始め……。

## 特徴

- **業務ポータル・デザイン**: 事務的で冷徹なポータルサイトUI。Inter と Hiragino Mincho を組み合わせ、ビジネス的な機能美を演出。
- **時事ニュース同期 (RSS2JSON)**: Google News から最新のニュースタイトルを取得し、それを「社会的な汚染源」として解析プロセスに組み込みます。
- **Gemini AI による動的案件生成**: 
  - AIが各ステージの「清掃物件の詳細」を生成。
  - AIが「職員（プレイヤー）の冷ややかで皮肉な独り言」を生成し、没入感を高めます。
- **叙述的シミュレーター**: 固定観念を覆すストーリー展開と、最終的なシステムダウン演出。

## セットアップ & デプロイ方法

### 1. GitHub へのアップロード
以下の3つのファイルを GitHub リポジトリ（公開用）のルートにアップロードしてください。
- `index.html`
- `style.css`
- `main.js`

### 2. GitHub Pages の有効化
1. リポジトリの **Settings > Pages** を開きます。
2. **Branch** で `main` (または `master`) ブランチの `/ (root)` を選択し、**Save** をクリックします。

### 3. 使用方法
1. 公開された URL にアクセスします。
2. 職員ID、好物を入力し、「打刻：業務を開始」をクリック。
3. ケースごとに「漂白を開始する」→「洗浄する」→「次へ」のサイクルで業務を進めてください。

## 注意事項
- **セキュリティ**: API キーはブラウザの `localStorage` にのみ保存され、外部には送信されません。
- **プライバシー**: 入力した名前や好物は、AI解析と走馬灯演出にのみ使用されます。
