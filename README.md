# Memory Whitening System - 意識同期プロセス (Chronicle Edition)

職員用業務ポータル、および死後の記憶整理システム。本バージョンでは、「当日（生存）」と「翌日（死後）」の二部構成で物語の完結を描きます。

## コンセプト

「清潔な社会」の維持のため、あなたは「職員」として他者の記憶を漂白する業務に従事します。しかし、業務の果てに待っているのは、あなた自身の記憶を漂白する作業です。

## 二部構成の体験

### 第1部：当日（生存時） - 業務ポータル
- **クリック漂白パズル**: 提示される解析データの中から「不潔な単語」をクリックして浄化します。
- **サトウとのチャット**: 同僚のサトウとリアルタイムで会話をしながら業務を進めます。
- **クライマックス**: 業務終了後、不可避な「急な衝撃」によって第一部は幕を閉じます。

### 第2部：翌日（死後） - 記憶の整理
- **脳内メーカー風UI**: 画面が明るくなると、あなたの脳内が視覚化（脳内メーカー風）されています。
- **Gemini による私的な記憶チップ**: 
  - あなたが第1部で体験したこと、好物、サトウとの会話内容に基づき、AIが「あなたの最後の記憶」をチップとして10個生成します。
  - チップをタップして「ポフッ」と消去していくことで、あなた自身をこの世界から完全に浄化します。
- **浄化の果て**: すべての記憶を消し去った時、真の安らぎ（PURIFIED）へと至ります。

## セットアップ & デプロイ方法

### 1. GitHub へのアップロード
以下のファイルを GitHub リポジトリのルートにアップロードしてください。
- `index.html`
- `style.css`
- `main.js`

### 2. GitHub Pages の有効化
1. リポジトリの **Settings > Pages** を開きます。
2. **Branch** で `main` を選択し、**Save** をクリックします。
