# 記憶清浄適性検査 (Memory Whitening System)

厚生労働省 精神衛生管理局による、個体の精神的純度を測定するための適性検査システムです。

## コンセプト

「清潔な社会」の維持のため、個体の記憶に癒着した不要な執着（SNS、食欲、生理的記憶など）をスキャンし、漂白（Whitening）を行います。

## 特徴

- **公的ミニマリズムのデザイン**: 行政書類のような厳格さと異常さを、美しい余白と明朝体（Sawarabi Mincho）で表現。
- **Gemini AI による鑑定**: あなたが入力した「SNSの単語」や「食べ物」を、AIが公的機関の報告書スタイルで「不純物」として冷徹に分析します。
- **シミュレーションモード**: Gemini APIキーをお持ちでない場合、または通信エラー時は、自動的にプリセットされた解析シナリオが実行されます。
- **劇的なストーリー展開**: 検査の果てに待ち受ける、個体の「真実」についての叙述的演出。

## セットアップ & デプロイ方法

### 1. GitHub へのアップロード
以下のファイルを GitHub リポジトリ（公開用）のルートにアップロードしてください。
- `index.html`
- `style.css`
- `main.js`

### 2. GitHub Pages の有効化
1. リポジトリの **Settings > Pages** を開きます。
2. **Branch** で `main` ブランチの `/ (root)` を選択し、**Save** をクリックします。

### 3. 使用方法
1. 公開された URL にアクセスします。
2. Gemini API を使用する場合、ブラウザのコンソールまたは設定から `localStorage.setItem('gemini_api_key', 'YOUR_KEY')` を実行するか、必要に応じてキー入力画面をご利用ください。
3. 3つの質問に答え、漂白プロセスを開始してください。

## 注意事項
- このアプリは静的サイトとして動作します。入力された情報はサーバーには送信されず、AI解析もクライアントサイドで行われます。
- 物語上の演出であり、実際の行政機関とは一切関係ありません。
