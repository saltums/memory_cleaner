# Memory Whitening System - 業務ポータル (Interactive Edition)

職員用意識同期ポータル。本バージョンでは、「クリックによる漂白作業」と「同僚とのリアルタイムチャット」を組み合わせた、よりインタラクティブな体験を提供します。

## コンセプト

「清潔な社会」の維持のため、あなたは「職員」として個体の深層意識から不要な執着を除去する業務に従事します。テキストの中に潜む「不潔な単語」を見つけ出し、クリックして浄化してください。作業中、同僚のサトウからチャットが入ることもありますが、惑わされずに業務を遂行してください。

## 特徴

- **クリック漂白パズル**: 
  - システムが提示する解析データの中から、強調された「不潔な単語」をすべてクリックして漂白（Whitening）してください。
  - すべての単語を浄化するまで、次の案件へ進むことはできません。
- **同僚サトウとのチャット機能**: 
  - 画面右側のチャットエリアに、同僚のサトウからメッセージが届きます。
  - 世間話から始まり、次第に物語の核心や「外の世界」の異変を予感させる内容へと変化します。
- **Gemini AI による動的コンテンツ生成**: 
  - AIが「解析データ」を生成し、その中から適切な「漂白対象の単語」を選定してタグ付けします。
  - サトウのチャット内容も、AIがストーリーの進行に合わせて動的に生成します。
- **2画面分割の業務UI**: 事務的な作業エリアと、プライベートなチャットエリアを分けた臨場感のあるレイアウト。

## セットアップ & デプロイ方法

### 1. GitHub へのアップロード
以下のファイルを GitHub リポジトリ（公開用）のルートにアップロードしてください。
- `index.html`
- `style.css`
- `main.js`

### 2. GitHub Pages の有効化
1. リポジトリの **Settings > Pages** を開きます。
2. **Branch** で `main` ブランチの `/ (root)` を選択し、**Save** をクリックします。

### 3. 使用方法
1. 情報を入力し、「業務を開始」をクリック。
2. 強調された単語をクリックして漂白します。
3. チャットログを確認しながら、漂白が完了したら「次へ進む」をクリックしてください。
