# Memory Whitening System - 業務ポータル (Novel Game Edition)

職員用意識同期ポータル。本バージョンでは、物語を読み進めるノベルゲーム形式を採用しています。

## コンセプト

「清潔な社会」の維持のため、あなたは「職員」として個体の深層意識から不要な執着を除去する業務に従事します。解析されるデータに対し、あなたの内面（モノローグ）がどのように反応していくか。クリックで物語の進行を手導し、真相へと辿り着いてください。

## 特徴

- **ノベルゲーム・インターフェース**: 
  - **上画面（システム）**: 冷徹な解析情報が提示される。
  - **下画面（モノローグ窓）**: 職員の心の声が、情報の提示に合わせて逐次表示される。
- **クリック進行エンジン**: 
  - 一括表示ではなく、ユーザーの操作（クリック・タップ）によってテキストが1行ずつ進みます。
  - タイピング中のクリックによる「文章の全表示（スキップ）」機能を搭載。
- **Gemini による対話台本生成**: 
  - AIが「解析データ」と「職員の反応」をセットにした「台本」を動的に生成します。
  - ニュースや個人の嗜好に基づいた、唯一無二の物語が展開されます。
- **時事ニュース連携**: 現実のニュースを汚染源として取り込む RSS 同期機能。

## セットアップ & デプロイ方法

### 1. GitHub へのアップロード
以下のファイルを GitHub リポジトリのルートにアップロードしてください。
- `index.html`
- `style.css`
- `main.js`

### 2. GitHub Pages の有効化
1. リポジトリの **Settings > Pages** を開きます。
2. **Branch** で `main` を選択し、**Save**。

### 3. 使用方法
1. 情報を入力し、「業務を開始」をクリック。
2. **画面全体をクリック**して、テキストを読み進めてください。演出が終わると次の案件へと進みます。
